Tools:

We have used the following to design and deploy the blog:

  • HTML/CSS http://www.w3schools.com/css/ Site designed using only HTML/CSS
  • Pivotal Tracker https://moqups.com/ Project management for user stories
  • Moqups https://moqups.com/ Online software for creating detailed wireframes
  • Github https://github.com/ Repository/hosting
  • Harp http://harpjs.com/ Creating static site Harp requires the use of Node.js, Node Package Manager and Harp.js see https://github.com/nofootnotes/harp-git
  • QUnit http://qunitjs.com/ Unit testing
